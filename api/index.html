目前OpenAI已经放开了ChatGPT账号的注册流程，不需要手机号就可以成功注册和使用ChatGPT了。 但是申请 ChatGPT KEY不仅需要验证手机号，还需要绑卡。

很多人因为卡或者是网络原因导致绑卡失败，还容易导致账号被OpenAI禁用。

wildcard 就是专门针对 ChatGPT做了优化的虚拟银行卡，很方便注册升级和绑卡。

我通过 wildcard 使用了一年，账号几乎没有出现任何问题。

注册 wildcard

注册 Wildcard 流程也很简单，几乎1分钟就可以完成注册。

PS：通过官网邀请码 GOGPT，可以优惠2美元！

记得一定要选择 WildCard虚拟卡选项，否则后面无法通过虚拟银行卡绑定到OpenAI上。

按照要求填写手机号注册之后，会跳到支付宝进行身份验证，通过之后，就需要支付9.9$的费用开一个虚拟信用卡。

后续这张卡就是安全独立的，几乎不会出现封号的情况。

wildcard 虚拟银行卡

成功注册之后，就会得到下面一张类似的虚拟银行卡和账单地址信息。

可以按照你订阅的服务进行充值和提现。

充值之后去订阅GPT之后，如果出现了问题可以自动退费，然后可以提现的哈

所以使用 Wildcard 基本山不会有任何的损失。

记得提前向银行卡充值，后面OpenAI绑卡的时候会直接通过银行卡扣款，否则就会绑卡失败

登录OpenAI

网址：https://platform.openai.com/

输入ChatGPT账号密码登录即可。

绑定付款方式

直接访问绑卡地址：https://platform.openai.com/account/billing/overview

在Billing栏目中选择 Add payment details 进行绑卡。

点击“Set up paid account”，选择是个人使用还是公司。

一般都是选择个人，选择 Individual 即可。

OpenAI绑卡

输入卡号，有效期、CVV以及邮编，下面输入姓名、WildCard 提供的帐单地址，然后点“continue“。

绑卡结束后，会让你选择充值金额，可以是5美元到50美元之间的任意金额。为了避免出现问题，先小额充值5美元。

点击 Confirm payment，即可完成绑卡操作。

申请API Key

成功绑卡完毕之后，就可以申请 API Key

直接申请链接： https://platform.openai.com/api-keys

申请API Key，需要验证手机号。

登录 WildCard：https://wildcard.com.cn/i/GOGPT，在页面里面找到「海外手机号」服务，点击「申请手机号」，

申请成功之后，复制分配的手机号。

国家请选择 🇬🇧 英国 United Kindom，然后将号码粘贴到OpenAI的注册页面输入框里，点击下方的“发送验证码”按钮。

在 WildCard 页面 点击刷新按钮，知道看到6位数验证码已经显示在网页上了，复制然后回到 OpenAI 填写，就可以完成OpenAI注册过程。

如果长时间没有收到验证码或者提示手机号码已经被注册，就申请新手机号，一般3个号码能稳定收到至少1个验证码。

验证成功之后会有提示，这里请注意，每个海外手机号可以验证两个 API，如果第二次使用这个手机号验证，会提示这个手机号已经绑定了一个 API，这个没有影响，可以正常使用，无需担心。

然后正常创建和复制 api key 就行了

查看API的消费情况

能够消费的最大金额就是充值金额。比如前面我们充值了5美元，那么最多我们只能使用5美元。

需要额外注意的是：

如果连续2次扣费失败（比如遇到卡内余额不足），OpenAI 就会封禁这个开发者帐户和对应的卡号。

所以一定要保持卡内余额充值，当卡内余额不足时，OpenAI也可以发邮件提醒，所以记得查邮件。

消费金额查看：https://platform.openai.com/usage

不同的API模型的价格是不一样的。

模型价格：https://openai.com/pricing

ChatGPT4 API的价格比ChatGPT3.5 API的价格贵了很多，但是功能也更加强大。大家可以根据实际的需要选择合适的模型。
